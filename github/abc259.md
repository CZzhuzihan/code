## [AtCoder Beginner Contest 259](https://atcoder.jp/contests/abc259)
### [A - Growth Record](https://atcoder.jp/contests/abc259/tasks/abc259_a)
#### 题目翻译
`Takahashi`的`n`岁生日时有`t`厘米高,他在`x`岁之前每年长`d`厘米,问他`m`岁时多高.
#### AC代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,m,x,t,d;
	cin>>n>>m>>x>>t>>d;
	cout<<t-d*max(x-m,0);
	return 0;
}
```
